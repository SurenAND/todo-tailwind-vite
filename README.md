This is a Todo App project with [`Tailwindcss`](https://tailwindcss.com/) and [`ViteJs`](https://vitejs.dev/) Tool and [`JSON Server`](https://www.npmjs.com/package/json-server).

## Getting Started

First, install packages:

```bash
npm install

```
